CentOS安装Node.js

发布 : 2019-02-13 分类 : 运维 浏览 :

一、下载node.js安装包

下载地址: http://nodejs.cn/download/
根据自己服务器Linux系统版本,选择64位或者32位安装包.

例如我下载的安装包为: node-v10.5.0-linux-x64.tar.xz

二、上传安装包到服务器

使用rz命令上传(rz具体使用[参考CentOS安装JDK.md])

三、解压安装包

安装包为.tar.xz结尾,需要二次解压.
第一步: xz -d node-v10.5.0-linux-x64.tar.xz
第二步: tar xvf node-v10.5.0-linux-x64.tar

四、导出node.js命令[全局配置]

编辑[profile]
命令: vi /etc/profile

配置node.js命令[注意: 路径要写成自己node.js的路径]
# node.js
export NODE_HOME=/home/sdk/node-v10.5.0-linux-x64
export PATH=${NODE_HOME}/bin:$PATH 

编辑完成后,使用[source profile]使profile文件生效.

五、测试

在终端输入[node-v],若可打印出node的版本信息,表示安装和配置已经成功.
